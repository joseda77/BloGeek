# BloGeek

Desarrollado por Jose David Tello y Juan Camilo Morales

Aplicación web para compartir contenido por medio de sockets,esta es una aplicación desarrollada en Node JS (BackEnd) y React JS (FrontEnd), usando frameworks como nodemon, express, mongoose.

Frameworks y librerias instaladas:

- Para instalar Express: npm install express --save
- Para instalar Mogoose: npm install mongoose --save
- Para instalar Nodemon: npm install nodemon --save-dev
- Para instalar Path: npm install path --save
